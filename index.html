HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML is used to create a structure for a website.
  • Using different elements that work with each other to display content for the website.
  • Common element types for HTML include: Meta-related elements, Content division elements, and Semantic elements.

CSS

  • CSS means Cascading Style Sheets and is used to give the webstie a style.
  • CSS is used to define the aesthetics of a website, along with organization.
  • It is used to give style rules for the HTML elements of the page
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

GIT

  • Git status: checks what branch we are currently on.
  • Git checkout -b branch-name: creats a new branch and switches to it.
  • Git add -A: adds all changes to the current staging area.
  • Git commit: commits all of the added changes from Git add -A to the code.
  • Git commit -m "add starter code": indicates which area of the file is being commited to change.

JAVASCRIPT

  • Javascript is used to give your website specific functions
  • Functions are instructions that tell the computer how to perform a task that it is asked to do.
  • These functions are reusable code.
  • Curly brackets {} are used to contain code for a specific task to be performed.
  • Examples of functions are IF statements, and FOR loops.
